Branching page

Why is making a branch important?

There are several reasons as to why it is vital to make branches, some of which are :

It makes your code base safe

Creating a branch would enable us to test our ideas. It gives us a copy of the code, wherein we can test our ideas without affecting the main source code

It is a good way to organize work

It is recommended to create a branch for a specific task may it be a feature, bug fix or even ideas that we want to experiment on!Think of a branch as a record of what was done and why

Makes collaboration easier

It allows multiple developers to contribute without conflicts. It prohibits every member from working on the same code directly, which can potentially leads to things getting messy. By creating a branch, each person has their own work environment where they can add or change features and later on merge to the main source code after testing.

Easier to test

It is easy for the team to test changes in a branch before merging it into the main source code. This is important to easily detect mistakes early , and avoid a huge mess in the main project.

Remote branches let others see your work

As a local branch is just on ones computer. Creating a remote branch by pushing the local branch to a remote branch like GitHub allows the other members of the team or others who are interested to see it and test it , and who knows, even contribute to it.

What are the git commands to create and close a branch?

To create a new local branch from main / develop branch. It is always a good idea to type git checkout main or git checkout develop , to ensure that you will be creating a local branch from there.

To create a new branch locally use

In your terminal, write:

git branch nameofthebranch

or

git checkout -b nameofthebranch

to create and switch to this newly created branch.

To close or delete a local branch

In your terminal, write:

git branch -d nameofthebranch

one might need to use

-D

if there are unmerged changes.

To close or delete a remote branch

In your terminal, write:

git push origin --delete nameofthebranch
